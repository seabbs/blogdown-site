Introduction to getTBinR

Abstract

A talk outlining the motivation behind, and core functionality for the getTBinR R package. Documentation, unit testing and the advantages of the R package structure were touched upon. The talk served as a launching point for a group discussion on possible additional datasets to add to the package and intesting analysis questions to investigate using the package.
